✈️ 출국 전 필수 설정

발음 재생(TTS)은 기기의 베트남어 음성 데이터를 사용합니다. 출국 전에 꼭 다운로드해 두면 비행기 모드나 데이터 로밍 없이도 발음이 나옵니다.

📱 iPhone (iOS)

  1. 설정 → 손쉬운 사용 → 말하기 콘텐츠
  2. 음성 → 베트남어(Tiếng Việt) 선택
  3. 기본 음성 대신 향상된(Enhanced) 또는 Premium 표시가 있는 음성(예: Linh)을 다운로드 — 훨씬 자연스러움

🤖 Android

  1. 설정 → 시스템 → 언어 및 입력 → 텍스트 음성 변환 출력
  2. 엔진을 Google TTS로 설정 (기본보다 훨씬 자연스러움)
  3. 설정 아이콘 → 음성 데이터 설치 → 베트남어 (Tiếng Việt) 다운로드

💻 Mac

  1. 시스템 설정 → 손쉬운 사용 → 말하기 콘텐츠 → 시스템 음성 → 관리…
  2. Tiếng Việt(베트남어) 중 Premium 또는 Enhanced 표시가 있는 음성 체크 → 다운로드

다운로드 후 브라우저를 완전히 종료하고 다시 실행해야 새 음성이 인식됩니다. 상단의 음성 선택 드롭다운이 보이면 ⭐ 표시가 있는 음성을 고르세요 (자동으로 기본 선택됨).

🔑 번역 기능 API 키 설정 (입력/듣기용)

하단 입력/듣기 버튼은 Google Cloud Translation API를 사용합니다. 브라우저용 API 키를 한 번만 발급받아 앱에 저장하면 됩니다.

  1. Google Cloud Console에서 Cloud Translation API 활성화 (API 및 서비스 → 라이브러리 → "Translation API" 검색 → 사용 설정)
  2. 새 API 키 생성 (사용자 인증 정보 → + 만들기 → API 키) — MP3 생성용과 다른 키를 써야 보안 분리됨
  3. 발급된 키의 "키 제한" 설정:
    • 애플리케이션 제한사항 → "HTTP 리퍼러(웹사이트)" → https://jjackddo.github.io/* 추가 (로컬 테스트용으로 http://localhost:*도 추가)
    • API 제한사항 → "키 제한" → Cloud Translation API만 체크 → 저장
  4. 키를 복사해서 입력 또는 듣기 버튼을 처음 눌렀을 때 뜨는 입력창에 붙여넣고 저장

키는 브라우저의 localStorage에만 저장되며 GitHub에 올라가지 않습니다. 리퍼러 제한 덕분에 혹시 유출되더라도 jjackddo.github.io 외 다른 곳에서는 사용 불가능합니다.

🎙️ 고품질 음성(MP3) 사용하기

Google Cloud TTS로 미리 만든 MP3가 있으면 기기 음성 대신 원어민급 Neural 음성이 재생됩니다. 상단의 음성 상태 표시가 🔊 고품질 음성 N개로 바뀌면 활성화된 것입니다.

MP3를 만들려면 Mac 터미널에서:

  1. Google Cloud 콘솔에서 Text-to-Speech API 활성화 + API 키 발급
  2. 프로젝트 루트 .env 파일에 GOOGLE_TTS_API_KEY=... 저장
  3. python3 tools/generate_audio.py 실행

자세한 순서는 대화 가이드 참고. 생성된 MP3는 audio/ 폴더에 저장됩니다.

📥 비행기 모드에서도 쓰려면

아래 버튼을 누르면 모든 MP3를 브라우저 캐시로 미리 받아둡니다 (한 번만 누르면 됨, WiFi에서 권장).

📲 홈 화면에 아이콘 추가

  • iPhone Safari: 공유 버튼(네모+↑) → "홈 화면에 추가"
  • Android Chrome: 메뉴(⋮) → "홈 화면에 추가"

한 번 추가한 뒤에는 비행기 모드에서도 앱처럼 열립니다.

🔊 사용 방법

  • 문장 카드를 탭하면 베트남어 발음이 재생됩니다.
  • 상단 검색창에 한국어 또는 베트남어를 입력해 빠르게 찾을 수 있습니다.
  • 속도 슬라이더로 발음 속도를 조절하세요 (0.7~0.8x가 따라 하기 좋음).
  • 상단 카테고리 탭을 누르면 해당 섹션으로 이동합니다.

❓ 소리가 안 나요

  • 무음 모드(iPhone 좌측 스위치) 해제 & 볼륨 확인
  • 첫 재생은 반드시 화면을 탭한 뒤 동작 (브라우저 자동재생 정책)
  • 베트남어 음성이 기기에 설치되어 있지 않으면 다른 언어 음성으로 들릴 수 있음 → 위 1번 확인
  • 기내/로밍 없는 상태에서 재생 안 되면 OS 음성 데이터가 아직 다운로드되지 않은 것
