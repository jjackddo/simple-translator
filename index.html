🇻🇳 베트남 여행 회화

항목을 눌러 베트남어 발음을 들어보세요

✍️ 한국어 → 베트남어

🎤 듣기 → 한국어 번역

도움말

🔊 사용 방법

  • 문장 카드를 탭하면 베트남어 발음이 재생됩니다.
  • 상단 검색창에 한국어나 베트남어를 입력해 빠르게 찾을 수 있습니다.
  • 카테고리 탭을 누르면 해당 섹션으로 이동합니다.
  • 하단 ✍️ 입력: 말하고 싶은 한국어를 써서 베트남어로 통역·재생.
  • 하단 🎤 듣기: 상대방 말을 녹음해 한국어로 번역해줍니다. 마이크 권한이 필요합니다.

✈️ 출국 전 준비

미리 해두면 비행기 모드나 로밍 없이도 앱이 동작합니다.

  1. 베트남어 음성 데이터 다운로드
    • iPhone: 설정 → 손쉬운 사용 → 말하기 콘텐츠 → 음성 → Tiếng Việt → 향상된(Enhanced) 음성
    • Android: 설정 → 시스템 → 텍스트 음성 변환 → Google TTS → 베트남어 데이터 설치
  2. 홈 화면에 아이콘 추가
    • iPhone Safari: 공유(네모+↑) → 홈 화면에 추가
    • Android Chrome: 메뉴(⋮) → 홈 화면에 추가
  3. 오프라인용 음성 미리 받기 (아래 버튼, WiFi에서 한 번만)

입력·듣기 기능은 인터넷이 필요합니다. 오프라인에서는 저장된 회화만 동작합니다.

❓ 소리가 안 나요

  • 무음 모드(iPhone 좌측 스위치) 해제 · 볼륨 확인
  • 첫 재생은 반드시 화면을 한 번 탭한 뒤에 동작합니다 (자동재생 제한).
  • 오프라인에서 안 들리면 베트남어 OS 음성이 아직 다운로드되지 않은 것 → 위 "출국 전 준비" 참고.

⚠️ 번역 주의

문장은 초안이고 자동 번역이라 상황에 따라 뉘앙스가 다를 수 있습니다. 중요한 상황에서는 파파고나 Google 번역으로 한 번 더 확인하세요.
